## 📖 Overview

This project showcases a clean and modern sidebar navigation menu that can be integrated into websites, portfolios, admin dashboards, and web applications.
The goal of this project is to practice frontend development fundamentals while creating a reusable UI component.


## ✨ Features

✔️ Responsive Sidebar Layout

✔️ Clean & Modern User Interface

✔️ Smooth Hover Effects

✔️ Easy Customization

✔️ Lightweight and Fast

✔️ Pure HTML & CSS
